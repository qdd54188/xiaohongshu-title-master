# 小红书爆款标题生成器

一个 AI Agent Skill（Claude Code / Codex / Cursor 通用），输入内容和领域，自动生成全套小红书笔记包装方案。

## 能做什么

- **A/B 双策略标题**：稳妥流量型 + 情绪爆发型，各 5 个，覆盖数字、悬念、反差、恐吓等 7 种模板
- **封面拍摄指南**：构图类型、拍摄角度、色调方向、画面元素、封面文字、字体颜色，全部给到位
- **竞品套路拆解**：同类爆款标题的共性分析 + 你的差异化切入建议
- **正文钩子模板**：前三句怎么写，让读者点进来就不想划走
- **运营建议**：发布时间 + 三层标签策略 + 评论区引导话术
- **标题公式库**：10 个拿来就用的标题公式 + 8 种封面构图公式
- **A/B 测试规则**：怎么发、怎么看数据、什么时候删掉重发

## 安装使用

将 `SKILL.md` 放到 Claude Code / Codex / Cursor 的 skills 目录下即可。

```
Claude Code：~/.claude/skills/
Cursor：.cursor/skills/
Codex：.codex/skills/
```

## 使用示例

```
用户：帮我写小红书标题，美食领域，内容是"北京簋街5家必吃小龙虾测评"

输出：
  📊 内容分析 → 🔍 竞品热身 → 🅰️ A组5个标题 → 🅱️ B组5个标题
  → 🥇 A/B实战建议 → 📸 封面拍摄指南 → ✍️ 正文钩子 → 💡 运营包
```
