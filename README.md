# 小红书爆款标题生成器

一个 AI Agent Skill，输入内容和领域，自动生成 10 个高互动率小红书标题 + 最佳发布时间 + 爆款概率分析。

## 功能

- 支持 10+ 品类：美妆、旅行、美食、职场、穿搭、健身、家居、母婴、科技、情感
- 7 种标题类型自动覆盖：数字型、悬念型、情绪型、反差型、利益型、恐吓型、对比型
- TOP 3 精选推荐 + 爆款理由分析
- 发布时间建议 + 互动率预测
- 运营建议：封面图风格、正文结构、标签组合

## 安装使用

### 方法一：Claude Code / Codex

```
# 在 Claude Code 中直接使用
将 SKILL.md 放到 .claude/skills/ 目录下
```

### 方法二：腾讯 SkillHub

在 SkillHub 搜索「小红书爆款标题生成器」安装即可。

### 方法三：QClaw

```
skillhub install xiaohongshu-title-master
```

## 使用示例

```
用户：帮我写小红书标题，美食领域，内容是"北京簋街5家必吃小龙虾测评"

输出：
  📊 内容分析 → 10个标题 → TOP 3精选 → 运营建议
```
