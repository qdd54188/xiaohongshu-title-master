## 🤔 为什么做这个

我是小红书重度用户。过去一年最大的感受是：

> 正文写半小时，标题想一小时。封面怎么拍、正文前三句怎么写、什么时间发——这些比内容本身更让人头疼。

市面上有很多「标题生成器」，但它们只给你几个标题，拍屁股就走了。没人告诉你：**同一个内容，应该准备两套标题去 A/B 测试；标题和封面是强绑定的；不同时间段发同一篇笔记，数据能差 3 倍。**

我把这些经验写成了一个 AI Skill。不是让你多一个工具，是让你少一个烦恼。

---

## 🎯 它帮你做什么

```
你说：帮我写小红书标题，美食领域，簋街小龙虾测评

它输出：
  📊 你这条内容的核心卖点是什么
  🔍 同类爆款笔记用了什么套路（3个例子 + 共性分析）
  🅰️ A组标题：5个「稳妥流量型」— 收藏向、长尾流量
  🅱️ B组标题：5个「情绪爆发型」— 争议向、短期冲量
  🥇 实战建议：先发哪组，数据不好怎么换，多久看数据
  📸 封面拍摄指南：构图/角度/色调/画面元素/加什么字/字放哪/为什么
  ✍️ 正文前三句：怎么写才能让人不划走
  💡 运营包：几点发 + 三层标签 + 评论区引导 + 要不要投薯条
```

**不是给你 10 个标题然后说「挑一个吧」。是给你一整套打仗方案。**

## 📦 内含

| 模块 | 说明 |
|------|------|
| 10 种标题公式 | 数字+结果、身份+痛点、对比暴击… 拿来就用 |
| 8 种封面构图 | 俯拍平铺、人物手持、对镜自拍… 每种适合什么领域 |
| A/B 测试规则 | 两组标题怎么发、怎么看数据、什么时候重发 |
| 发布时间表 | 分平台、分时段、分工作日和周末 |
| 三层标签策略 | 热门标签 + 精准标签 + 长尾标签怎么搭配 |
| 敏感词黑名单 | 哪些词会触发审核被限流 |
| 封面字数规范 | 小红书 1080px 宽度下，加多少字、多大字号 |

## ⚡ 快速开始

把 `SKILL.md` 复制到你的 AI 编码工具 skills 目录：

```
# Claude Code
cp SKILL.md ~/.claude/skills/

# Cursor
cp SKILL.md .cursor/skills/

# Codex
cp SKILL.md .codex/skills/
```

然后对 AI 说：**「帮我写小红书标题，XX领域，内容是XX」**
